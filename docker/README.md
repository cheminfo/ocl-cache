Cache molecule directed information in sqlite3.

## Installation

`$ npm i ocl-cache`

On silicon mac you may need:

```
cd node_modules/better-sqlite3
npm run build-release
```

## Usage

```js
import { myModule } from 'ocl-cache';

const result = myModule(args);
// result is ...
```
